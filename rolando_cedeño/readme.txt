El proyecto esta desarrollado el backend en Laravel y el fronted en Vue y la base de datos en MYSQL
*BACKEND
	*Ubicarse dentro de la carpeta backend con la consola de comando
	*Ejecutar el comando composer install para instalar las dependencias
	*Ejecutar el comando php artisan key:generate
	*Ejecutar el comando php artisan serve

*FRONTED
	*Ubicarse dentro de la carpeta fronted con la consola de comando
	*Ejecutar el comando npm install para instalar las dependencias
	*Ejecutar el comando vue serve

*BASE DE DATOS
	*Restaurar la base de datos que se encuentra dentro de la carpeta base de datos
